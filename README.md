## Usage

To run a solution:

```sh
$ python3 -m [PROBLEM_NUMBER].py
$ cat [PROBLEM_NUMBER].clj | clj
$ rustc [PROBLEM_NUMBER].rs; ./[PROBLEM_NUMBER]; rm -rf [PROBLEM_NUMBER]
```
